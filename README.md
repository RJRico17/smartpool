# smartpool

Smart pool is a website created by RJ Fredrick Rico made to help track and organize cue data.

## how to run

install Node.js, nmp, and express

`npm init -y`

modify package.json as:

`"type": "module",`

install express

`npm install express`

run nodemon for constant `app.js` reloading

`npx nodemon`

to stop nodemon

`ctrl + c`
